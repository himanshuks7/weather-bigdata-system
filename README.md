# weather-bigdata-system
A distributed weather analytics system that collects real-time weather data,
processes it using Kafka and Spark Streaming, stores results in HDFS, and
visualizes insights through Streamlit dashboards.

## Tech Stack
- Python
- Apache Kafka
- Apache Spark
- HDFS
- PyTorch LSTM
- Streamlit

## Features
- Real-time weather ingestion
- PM2.5 prediction
- Interactive dashboards
- Historical analytics

## Architecture
OpenWeather API → Kafka → Spark Streaming → HDFS → Streamlit Dashboard
